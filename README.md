## 🚀 Featured Projects

### 🛡️ [LLM-Powered SSH Honeypot](https://github.com/abdelrahman-shafie/Grad-LLM-Honeypot)

High-interaction SSH honeypot deployed on AWS EC2. Hybrid architecture with deterministic handlers for fingerprint-sensitive commands and a fine-tuned 1.5B LLM with BM25 RAG grounding for dynamic response generation. Features a stateful virtual filesystem, coherent OS persona with anti-fingerprinting, and automated post-session threat intelligence reports via AWS Bedrock (Claude Sonnet).

`Python` `Paramiko` `llama.cpp` `BM25 RAG` `AWS EC2` `AWS Bedrock`
